### 一. 下载博客git到本地
### 二. 安装 [hexo](https://hexo.io/zh-cn/)
### 三. 当前项目目录中安装主题及node依赖包

```shell
node install
git clone --depth 1 https://github.com/hexojs/hexo-theme-landscape themes/landscape
```

### 四. 测试使用

```shell
hexo server 
```
